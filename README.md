# Горячие клавиши
#### Выделить все: 
Ctrl+A 
#### Копировать: 
Ctrl+C 
#### Вставить: 
Ctrl+V
#### Сохранить: 
ctrl+S
#### Вырезать: 
Ctrl+X
#### Оменить: 
Ctrl+Z 

# Работа с git
### Скачать проект: 
```bash
git clone <URL>
```
### учесть все файлы:
```bash
git add .
```
### Сохранить коммит: 
```bash
git commit -m "..."
```
### Отправить в облако:
```bash
git push
```

### Принть изменения из облака:
```bash
git pull
```
